## Password Management

### Changing Your Password

1. Log in to the admin dashboard with the current password
2. Click the **Settings** button in the header (or in the sidebar on mobile)
3. A password change panel will appear
4. Enter your current password, new password, and confirm the new password
5. Click "Update Password" to save

### Password Requirements
- Minimum 6 characters
- Must match in both fields

## Security Tips

1. Change the default password immediately after first login
2. Use a strong password with mixed characters
3. Never share your admin credentials
4. Log out when finished using the admin panel
5. Clear your browser cache if experiencing authentication issues

## Support

For technical issues or questions:
- Check the browser console for error messages
- Verify the admin password is correct
- Try clearing browser cache and cookies
- Ensure you're using the latest version of the admin panel
